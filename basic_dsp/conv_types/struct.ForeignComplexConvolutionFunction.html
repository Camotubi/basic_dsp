conv_data

The data which is passed to the window function

Actual data type is a const* c_void, but Rust doesn't allow that becaues it's usafe so we store it as usize and transmute it when necessary. Callers shoulds make very sure safety is guaranteed.

is_symmetric

Indicates whether this function is symmetric around 0 or not. Symmetry is defined as self.calc(x) == self.calc(-x).
